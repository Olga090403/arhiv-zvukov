# Landing moodboard — «Архив звуков»

> Синтез M.1 (Mobbin) + M.2 (Refero). Бренд: §10 product_book.md — минимализм + lo-fi/ретро, «архивный, но живой», `#F5F1E8` / `#0F0F12` / `#F2C94C` / `#7C5CFF`.

## Что мне нравится

- [YouTube Audio Library](https://mobbin.com/explore/screens/d089c828-40e2-4167-8cf4-32c165cb77b5) — минималистичный каталог с поиском и фильтрами сразу в hero; ощущение «бесплатного архива без барьера входа».
- [Record Club](https://mobbin.com/explore/screens/07cdf0f0-6d6e-4a4d-90fb-8c97ced5570a) — плейлистная подача «коллекции» звуков; кураторский архив, а не склад файлов.
- [Albumcolors](https://refero.design/styles/99be9294-9ab4-4533-8e44-ccc274dc99ab) — плакетный hero с oversized all-caps заголовком, tight letter-spacing и warm canvas; архив через типографику.
- [Say Briefly](https://refero.design/styles/2dab5b15-55c1-4835-8d8b-0d42f67604bc) — тёплая «бумажная» base + жёлтый highlight на ключевых словах; один энергичный акцент без перегруза.
- [Suno](https://refero.design/styles/9844e7bf-4bff-48e6-8efc-e45002ce5226) — dark full-bleed hero с grainy glow, floating media cards и одним ярким CTA; «студия», а не корпоративный SaaS.
- [Desktop.fm](https://refero.design/styles/5b70f172-2ba2-4f26-9956-251d4cfc0329) — один focal point по центру, lo-fi/retro digital vibe; игровая простота как у Chrome Music Lab.
- [Epidemic Sound — SFX landing](https://refero.design/pages/2306913e-12f9-441b-b6bc-0c52086f7d0f) — beige hero `#F2F2EA` + search bar + category grid; структура каталога (layout, не тон).
- [Epidemic Sound — Playlist View](https://mobbin.com/explore/screens/98d48604-ab2c-4aa8-b05b-c7781dc99d41) — фильтры mood / genre / duration / BPM; сильный паттерн discovery для блока features.
- [Descript — Subscription](https://mobbin.com/explore/screens/eaf086ee-c7aa-489c-a9bc-625a8d55909a) — freemium-модалка подписки без агрессивного upsell; референс для Premium $4.99.

## Что не делать

- Серый текст на сером фоне (плохой контраст, «мутный архив»).
- Stock-фотки с улыбающимися людьми (ломают атмосферу «забытых звуков»).
- 5 разных шрифтов (достаточно display + body + mono для тегов).
- Neon/club-эстетику целиком (Suno pink→yellow gradient как основной стиль).
- Корпоративный SaaS с pixel-иконками и «видеомейкерским» тоном (Epidemic marketing).
- Lifestyle-фото в hero вместо звуковых артефактов и waveform.
- Ghost-only кнопки без явного primary CTA («Случайный звук» / «Попробовать»).
- Enterprise AI-landing с sidebar workspace (ElevenLabs product UI, не marketing tone).

## 3 принципа моего лендинга

1. **Архив через типографику и звук, не через декор.** Плакетный hero (Albumcolors + Say Briefly): одно сильное обещание, много воздуха, 3 playable-примера или «Случайный звук» — без stock-иллюстраций.
2. **Тёплая бумага + один «ламповый» акcent.** Light: `#F5F1E8` + `#1A1A1F`; янтарь `#F2C94C` только на CTA и highlight-словах; фиолет `#7C5CFF` — зона микшера/Premium, не в hero.
3. **Живость = интерактив звука, не неон.** Энергию даёт playable preview, waveform и пресет Lo-fi за 20 секунд (Suno + Desktop.fm + YouTube Audio Library), при минимальном chrome и dark vinyl `#0F0F12` в dark mode.
